= Keystone代管服務採用Equinix技術
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
NetApp已與Equinix合作、在NetApp Keystone Equinix資料中心內代管各項功能、確保為您提供統一化的解決方案。

採用Equinix技術的Keystone代管（Co-Loo）服務與標準Keystone服務產品不變。

在本服務中：

* Equinix提供空間、電力、冷卻、網路及發票、 和儲存設備。
* 對某些Keystone銷售提案提供支援。
* 這項服務在11個國家/地區的21個資料中心均受到支援。
