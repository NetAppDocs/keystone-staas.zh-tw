= 取得 Keystone 的說明
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
NetApp 為 Keystone 客戶提供營運服務。客戶將獲得專屬的 Keystone Success Manager（KSM）、直接支援聯絡資訊以及透過 ITOM 解決方案實現的主動監控。



== NetApp 支援服務

NetApp 為 Keystone 客戶提供遠端維運服務。這些服務涵蓋一系列儲存管理活動，包括資產和組態管理、容量和效能管理、變更管理、事件和事故管理、服務要求履行以及報告。

.Keystone成功管理程式
您的 KSM 將與您緊密合作，負責您的 Keystone 服務，並提供每週或每月的帳單和營運報告。具體職責已在您的 NetApp Keystone 協議中明確規定。



== 聯絡支援部門

如果您在使用 Keystone 訂閱時遇到問題，請使用您的 NSS 憑證登入 link:https://support.netapp.com["NetApp 支援網站"^] 以建立支援案例。

如果您沒有 NSS 登入資訊，link:https://mysupport.netapp.com/site/user/registration["註冊您的帳戶"^]請使用訂閱中某個節點的序號（SN）。

如需附螢幕擷取畫面的逐步指示，請參閱 link:https://kb.netapp.com/nss/Support_Site/How_to_open_a_technical_support_case["如何開啟技術支援案例"^]。

如果您想直接與支援代表通話，請撥打以下電話號碼之一：

* 美國 / 加拿大：888 463 8277
* 歐洲：00 800 44 638277
* 印度：000 800 100 8948
* 更多全球數據：link:https://www.netapp.com/company/contact-us/support/["https://www.netapp.com/company/contact-us/support/"^]




== 支援監控

NetApp 使用 ITOM 監控解決方案主動監控您的產品和訂閱服務的運作狀況，並連線到 NetApp Keystone 環境進行故障排除。
